# Histórias de Usuários

## Introdução

Esta página reúne todas as histórias de usuários elicitadas para o sistema da **Agência Korus**. As histórias descrevem funcionalidades do ponto de vista dos diferentes perfis de usuário do sistema.

## Metodologia

As histórias foram organizadas em uma tabela única. Cada história segue o formato: *"Como [perfil], eu quero [ação] para [benefício]"*, e possui um identificador único (**ID**).

**Legenda:**

- **USxx** — História de Usuário nº xx

---

## Histórias de Usuários

A Tabela 1 apresenta todas as histórias de usuários elicitadas para o sistema da Agência Korus.

<center><b>Tabela 1</b> — Histórias de Usuários</center>

| ID   | História |
| ---- | -------- |
| US01 | Como usuário, eu quero me cadastrar e logar com e-mail e senha para acessar as áreas restritas do site de forma segura. |
| US02 | Como administrador, eu quero gerenciar os perfis de acesso para garantir que cada funcionário/usuário/cliente visualize apenas as funcionalidades do seu cargo. |
| US03 | Como usuário, eu quero escolher um serviço específico e ser direcionado a um formulário correspondente para que minha solicitação seja segmentada desde o início. |
| US04 | Como administrador, eu quero receber os dados dos formulários via e-mail ou WhatsApp para responder ao lead com a maior agilidade. |
| US05 | Como usuário do sistema (administrador, funcionário ou cliente), eu quero visualizar e interagir com projetos em formato de quadros (estilo Trello), com cards organizados por etapas e controle de permissões, para acompanhar o andamento dos contratos de forma clara, segura e colaborativa. |
| US06 | Como visitante do site, eu quero acessar uma landing page responsiva da Agência Korus que apresente informações institucionais, serviços, metodologia, portfólio e eventos, para entender a empresa, avaliar sua qualidade e navegar facilmente pelo conteúdo. |
| US07 | Como visitante do site, eu quero acessar uma área com e-books, cursos e materiais digitais da Agência Korus, para conhecer os conteúdos disponíveis e, caso tenha interesse, ser direcionado para a compra. |
| US08 | Como administrador, eu quero poder cadastrar, editar e remover serviços da agência, para manter a lista de serviços sempre atualizada conforme as necessidades do negócio. |
| US09 | Como administrador, eu quero publicar anúncios e comunicados em um feed direcionado a públicos específicos, para compartilhar informações relevantes de forma segmentada com funcionários, gerentes, clientes ou todos os usuários. |
| US10 | Como administrador, eu quero integrar o sistema com Google Analytics e Pixel Meta, para ter controle de métricas e acompanhamento de campanhas, permitindo analisar o desempenho e otimizar estratégias. |
| US11 | Como administrador, eu quero que o sistema possua um mecanismo de gamificação baseado em pontos (XP), para incentivar a produtividade e o engajamento dos funcionários nas atividades do projeto. E como funcionário, eu quero ganhar pontos ao iniciar e concluir tarefas, para acompanhar meu desempenho e evolução dentro do sistema. |
| US12 | Como usuário do sistema, eu quero acessar e aceitar os termos de uso, política de privacidade e configurações de cookies, para entender como meus dados são utilizados e garantir conformidade com a LGPD. |
| US13 | Como administrador, eu quero acessar um painel administrativo para cadastrar, excluir, editar usuários e alterar seus perfis, papéis e permissões, para controlar o acesso ao sistema de acordo com as responsabilidades de cada usuário. |
| US14 | Como usuário do sistema (cliente ou funcionário), eu quero que os projetos e atividades sejam organizados em uma Google Agenda ou agenda pessoal dentro do site, com datas de início e fim, para receber notificações e acompanhar prazos de forma organizada. |
| US15 | Como usuário do sistema, eu quero acessar uma área de "Meu Perfil" para atualizar minhas informações pessoais, como nome, foto de perfil e senha, para manter meus dados atualizados e garantir a segurança da minha conta. |
| US16 | Como administrador, eu quero cadastrar, editar e remover trabalhos, fotos, vídeos e coberturas de eventos no portfólio da agência, para exibir os resultados da empresa ao público de forma atualizada. |
| US17 | Como administrador, eu quero cadastrar, editar e remover e-books, cursos e materiais digitais, para manter o catálogo de conteúdos sempre atualizado. |
| US18 | Como administrador, eu quero visualizar, filtrar e organizar os leads captados por critérios como tipo de serviço, orçamento ou prioridade, para otimizar a triagem comercial. |
| US19 | Como cliente, eu quero visualizar um resumo do andamento do meu projeto na plataforma, para acompanhar a evolução do trabalho contratado sem depender de contato direto com a agência. |
| US20 | Como administrador, eu quero que o sistema ofereça suporte a uma integração futura com IA para triagem e qualificação automática de leads, para aumentar a eficiência comercial da agência conforme o sistema evolui. |
| US21 | Como usuário do sistema, eu quero receber notificações sobre eventos importantes (novos leads, alterações em cards, prazos próximos), para me manter atualizado sem precisar consultar a plataforma constantemente. |
